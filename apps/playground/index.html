01

Quick Start

One call per widget. Polite defaults: nothing auto-fires on load.

import { soft } from 'soft-feedback'
// No CSS import needed; styles live inside the widget's Shadow DOM.

soft.csat({
  trigger: { type: 'event', name: 'ticket_resolved', delayMs: 800 },
  onSubmit: (p) => fetch('/api/feedback', { method: 'POST', body: JSON.stringify(p) }),
})

// Fire the trigger whenever the moment is right:
soft.track('ticket_resolved')

Appearance Tokens

White-label by default. Override any --sf-* token through appearance.tokens; a single override re-skins the whole widget.

TokenDefault (light)What
color-accent#4f46e5Primary / interactive color
color-bg#ffffffCard background
color-fg#1a1a23Text color
color-surface#f6f7fbInset surfaces (options, inputs)
color-border#e6e7eeBorders
radius16pxCard corner radius

Built-in presets: minimal, soft, glass, high-contrast. Spread one into appearance.tokens.
